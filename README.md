## For Termux/Ubuntu/SSH User

```bash
pkg update && pkg upgrade
pkg install git -y
pkg install nodejs-lts
pkg install ffmpeg -y
git clone https://github.com/xziyyy/clairity.git
cd clairity
npm install
```

[RECOMMENDED INSTALL ON TERMUX]
```bash
pkg install yarn
yarn
```

Run
```bash
npm start
```
Editing on Termux
```bash
npm install mc -y
mc
```
